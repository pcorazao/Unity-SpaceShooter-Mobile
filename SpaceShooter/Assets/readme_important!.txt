In order to make this game run you need to download latest Vuforia SDK for Unity 
from https://developer.vuforia.com/resources/sdk/unity 
and import it to your project.

Find tutorial in root of the Assets folder explaining how this game was changed from original to Augmented reality.

To test the game open scene /Done/Done_Scenes/Done_Main

For more information about Augmented reality and Live Game Board, support and more tutorials visit http://developer.livegameboard.com/
